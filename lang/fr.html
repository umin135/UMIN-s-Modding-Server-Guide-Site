Ce document explique les prérequis pour les commissions, ce que je ne peux pas faire, les tarifs, et le fonctionnement du système de candidature et de file d'attente.

Merci de le lire avant de soumettre une demande de commission — cela facilitera grandement le processus.

Veuillez noter que même si ce document est disponible en plusieurs langues, toutes les communications durant le travail de commission se font uniquement en anglais.

1. Prérequis

  • Tous les mods commandés sont toujours distribués de la même manière que les autres mods publiés sur mon Patreon.
    Cela signifie que les commissions privées sont impossibles. Toutes les commissions sont d'abord publiées en accès anticipé, puis rendues publiques — comme tout autre mod.
  • Les mods NSFW ne sont pas disponibles.
    L'exposition jusqu'au niveau de Dead or Alive ou The First Descendant est le maximum autorisé.
    La représentation des organes génitaux masculins ou féminins est absolument impossible en toutes circonstances.
    (À titre informatif, les représentations NSFW explicites sont également une infraction pénale en Corée du Sud.)

2. Ce que je ne peux pas faire

Je ne peux pas créer de mods de visage avec des expressions faciales fonctionnelles.

Je suis capable de faire du face modding, mais je n'ai pas encore de flux de travail stable pour le livrer de manière fiable en tant que commission.

Le face modding est toujours expérimental pour moi, et je ne suis pas à un niveau où je me sens à l'aise pour le facturer.

Je ne peux pas créer de designs entièrement originaux.

Je ne peux pas travailler uniquement à partir de concept art ou de designs personnalisés purs. Au minimum, un modèle extrait d'un autre jeu est requis.

Cela dit, vous n'avez pas besoin de fournir le modèle extrait vous-même. Je localise et extrais généralement le modèle moi-même, je vous demande seulement de confirmer avec précision si un tel modèle existe.

Les mods de cheveux que je peux créer sont très limités.

Si la source utilise un mesh basé sur Haircard d'un jeu relativement récent — comme Stellar Blade ou Mortal Kombat 1 — je peux produire d'excellents résultats.

Cependant, les anciens styles Hair Card — comme ceux de Tekken 7 ou DOA5/6 — nécessitent la conception d'un nouveau mesh de cheveux entièrement de zéro. Cela prend beaucoup trop de temps, et je ne suis pas sûr de pouvoir livrer un résultat qui vaut le coup de facturer.

Gardez cela à l'esprit lorsque vous discutez de votre commission avec moi.

Je ne peux pas créer de mods de scène pour le moment.

Le stage modding est encore un domaine où j'expérimente et j'apprends beaucoup.

À la place, je prévois d'ouvrir les commissions de scène séparément à l'avenir — d'ici là, je n'accepte pas les demandes de commission de scène.

3. Tarifs

  • En général, 30–40 USD pour les mods de tenues/personnages, bien que le prix puisse varier selon la méthode et le temps requis.
  • Le prix exact est déterminé à l'avance, en tenant compte du temps estimé et de la méthode de production.
  • Le paiement est toujours effectué une fois que je détermine que plus de la moitié du travail est terminée.

4. Système de file d'attente

La file d'attente des commissions existante est devenue excessivement longue, causant des difficultés de traitement.
Pour remédier à cela, je passe à un système de nouvelle file d'attente courte et rotative.

Principe fondamental : La priorité de la file d'attente héritée est avant tout.

Ceux inscrits dans la file d'attente héritée seront toujours classés avant les nouveaux candidats, quel que soit l'ordre de candidature.

Les deux points clés sont :

  • La file d'attente n'accepte qu'un petit nombre de créneaux à la fois, permettant une rotation plus rapide.
  • La priorité des candidats précédents est entièrement préservée.

4.1 Contexte

  • La file d'attente est devenue si longue que de nombreuses demandes attendent depuis 6 mois à plus d'un an.
  • Les demandes plus anciennes sont plus susceptibles d'être injoignables, ce qui a souvent bloqué le processus même lorsque j'essayais de procéder dans l'ordre.
  • Les nouvelles candidatures sont actuellement fermées, et à l'avenir j'opérerai selon le nouveau système décrit ci-dessous.

4.2 Terminologie

TermeSignification
File héritéeLa liste des anciennes demandes non traitées soumises sous l'ancien système. Elle est conservée et non supprimée.
Nouvelle fileLa file d'attente courte et active où le vrai travail est traité. Seul un petit nombre de créneaux sont acceptés par tour, et le nombre est déterminé à chaque tour.
Avantage prioritaireUn avantage à usage unique permettant aux inscrits de la file héritée de se classer avant les nouveaux candidats dans la nouvelle file. Il est consommé une fois utilisé.

4.3 Règles de la nouvelle file

  • La nouvelle file ne fonctionne qu'avec un petit nombre de créneaux.
  • Elle s'ouvre et se ferme sur de courts cycles, maintenant une rotation rapide.
  • Une fois toutes les commissions actives terminées, le tour suivant s'ouvre. (→ 4.6 Cycle de fonctionnement)

Remarque — Le nombre de créneaux acceptés n'est pas fixe.

Le nombre « 5 » utilisé dans ce document est purement un exemple.

Le nombre de créneaux acceptés par tour peut être aussi peu que 2 ou 3, décidé à nouveau à chaque tour.

4.4 Système de priorité

Après ouverture de la nouvelle file et collecte des candidatures, la priorité est calculée comme suit.

Étape 1 — Attribuer le classement de base par ordre de candidature
Toutes les candidatures reçoivent un rang de base basé sur l'ordre dans lequel elles ont été soumises.

Étape 2 — Appliquer la priorité de la file héritée (préséance absolue)
Tout candidat qui était inscrit dans la file héritée est déplacé au-dessus de tous les nouveaux candidats, quel que soit le moment de sa candidature.

  • Les détenteurs de file héritée sont toujours classés avant les nouveaux candidats.
  • Parmi les détenteurs de file héritée, le rang est déterminé par leur position d'origine dans la file héritée.
  • Parmi les nouveaux candidats, le rang est déterminé par l'ordre de candidature.

L'ordre final est toujours : [Détenteurs de file héritée (par rang d'origine)] → [Nouveaux candidats (par ordre de candidature)].

Étape 3 — N'accepter que le nombre de créneaux fixé pour ce tour
De la liste triée, seul le nombre de créneaux fixé pour ce tour est accepté ; le reste est rejeté pour ce tour.
(Le nombre de créneaux n'est pas fixe et varie à chaque tour — voir la note dans 4.3 Règles de la nouvelle file.)
Le rejet ne vaut que pour le tour actuel ; vous pouvez vous réinscrire au prochain tour.

4.5 Avantage prioritaire de la file héritée

  • Ceux dans la file héritée bénéficient d'un rang plus élevé garanti lors de leur candidature à la nouvelle file.
  • Cependant, au moment où vous êtes officiellement accepté dans la nouvelle file grâce à cet avantage :
    • Vous êtes retiré de la file héritée.
    • L'avantage prioritaire est également consommé. (Usage unique)

En d'autres termes, la priorité est garantie « tant que vous restez dans la file héritée. » Une fois que vous passez à la nouvelle file, vous êtes traité comme n'importe quel autre candidat.

4.6 Cycle de fonctionnement

  1. La nouvelle file s'ouvre.
  2. Les candidatures sont collectées, et les créneaux acceptés pour ce tour sont déterminés selon 4.4 Système de priorité.
  3. Les commissions acceptées sont traitées.
  4. Une fois tout le travail actif terminé, retour à l'étape 1 et ouverture de la nouvelle file.

Répéter ce cycle permet d'éliminer progressivement la file héritée également.

4.7 Exemple

L'exemple ci-dessous suppose que le nombre de créneaux acceptés pour ce tour est 5.

Comme expliqué précédemment, ce nombre n'est pas fixe et peut varier — 2, 3, etc. — à chaque tour.

Supposons que la nouvelle file s'ouvre et que 10 personnes (A–J) postulent dans l'ordre.
Parmi elles, C, F et H étaient dans la file héritée, avec des rangs hérités de C (1er) · F (2e) · H (5e).

Résultat du calcul de priorité :

Rang finalCandidatRaison
1CFile héritée — 1er
2FFile héritée — 2e
3HFile héritée — 5e
4ANouveau candidat (a postulé en premier)
5BNouveau candidat
6~D, E, G, I, JNouveau candidat — rejeté ce tour

Résultat (en supposant 5 créneaux acceptés) :

  • Top 5 — C, F, H, A, B — sont acceptés.
  • C, F et H ont été acceptés dans la nouvelle file → retirés de la file héritée + avantage prioritaire consommé.
  • D, E, G, I, J sont rejetés ce tour (peuvent se réinscrire au prochain tour).
  • Une fois toutes ces commissions terminées → la nouvelle file du tour suivant s'ouvre.

Si le nombre de créneaux acceptés pour ce tour était 3, seuls les 3 premiers — C, F, H — seraient acceptés, et tout le monde à partir de A serait rejeté.

4.8 Questions fréquentes (FAQ)

Q. Je suis dans la file héritée — dois-je postuler séparément à la nouvelle file ?

Oui. Être dans la file héritée ne vous fait pas avancer automatiquement. Vous devez postuler à l'ouverture de la nouvelle file, moment auquel votre avantage prioritaire sera appliqué.

Q. Si je suis rejeté ce tour, ma priorité de file héritée disparaît-elle ?

Non. L'avantage n'est consommé que lorsque vous êtes officiellement accepté dans la nouvelle file. Si rejeté, vous restez dans la file héritée et pouvez utiliser l'avantage à nouveau au prochain tour.

Q. Combien de créneaux sont acceptés par tour ?

Ce n'est pas fixe. Cela varie tour par tour — généralement entre 2 et 5.

Q. Quand la file héritée sera-t-elle entièrement vidée ?

Elle sera progressivement vidée au fil des tours successifs. Cependant, les demandes où le candidat est injoignable peuvent être retardées.
